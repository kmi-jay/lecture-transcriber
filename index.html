● Groq API Key

영구 무료 — 카드 등록 불필요
console.groq.com/keys → 구글 가입 → Create API Key

● AssemblyAI API Key

가입 시 $50 무료 크레딧 — 카드 미등록 시 자동과금 없음
assemblyai.com/app → 가입 → 대시보드에서 API Key 복사
크레딧 소진 후 카드 미등록이면 단순 중단됨

● OpenAI API Key 유료

이 서비스만 유료 — 174분 강의 ≈ ₩1,500
platform.openai.com/api-keys → Create new secret key
반드시 월간 한도를 설정하세요 (예: $5)

● Gemini API Key 무료

무료 티어 제공 — Gemini 1.5 Flash, 하루 1,500건
aistudio.google.com/app/apikey → 구글 로그인 → Create API Key
요약에만 사용 (전사에는 사용 안 함)
LLM 요약 설정
요약에 사용할 LLM

두 가지 모두 무료. 자동 선택 시 Groq 키가 있으면 Groq 우선, 없으면 Gemini 사용.
