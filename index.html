전사 방법 (음성→텍스트)

Groq·AssemblyAI·Google·OpenAI = 전용 음성인식 | Gemini = AI 멀티모달 전사 | Claude = 음성 미지원

프로바이더별 전사 품질 차이 안내
Groq (Whisper large-v3) — 무료. 속도 빠름. 한국어+영어 혼합 강의 시 영어를 한국어로 음차할 수 있음 (예: audit→오디트). 다국어 모드 권장.
OpenAI (Whisper) — 유료(저렴). Groq와 동일 Whisper 모델. 음차 문제 동일. 다국어 모드 권장.
AssemblyAI — 무료 크레딧. 한국어 전용 모델 사용. 화자분리(diarization) 지원. 영어 용어 인식 우수하나 문장이 파편화될 수 있음.
Google (Chirp 2) — 유료. Plaud 앱과 동일 엔진. 한국어 인식력 우수. 타임스탐프 지원. 분당 ~₩1로 저렴.
Gemini — 무료. AI 멀티모달 방식. 타임스탬프 미지원. 긴 침묵 구간에서 환각 발생 가능. 프롬프트로 제어 가능.

💡 권장: 한영 혼합 강의는 다국어 모드 + 정제(LLM) 후처리를 함께 사용하면 음차→영어 복원 등 품질이 크게 향상됩니다.
📝 강의 주제 입력: 주제를 입력하면 전사·정제·요약 모든 단계에서 해당 분야의 전문 용어 인식률이 향상됩니다.

● Google Cloud Speech-to-Text API Key 유료

유료 — 분당 ~₩1 (Chirp 2 모델)
전사 전용 (후처리 불가). Plaud와 동일 엔진.
Google Cloud STT API 키 발급 방법
1. Google Cloud Console에 접속 (구글 계정 필요)
2. 좌상단 프로젝트 선택 → 새 프로젝트 생성
3. APIs & Services → 라이브러리
4. "Cloud Speech-to-Text API" 검색 → 활성화
5. APIs & Services → 사용자 인증 정보 → API 키 생성
6. 생성된 API 키를 위 입력란에 붙여넣기
💡 팁: API 키를 웹사이트 제한으로 설정하면 유출 시 피해가 제한됩니다.
